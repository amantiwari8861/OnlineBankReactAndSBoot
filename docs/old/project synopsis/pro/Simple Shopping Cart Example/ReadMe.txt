Title: Simple Shopping Cart Example
Description: a simple shopping cart application via servlets. First the form of the shop will be displayed from which the user will select the products of his/her choice and add items to cart. As soon as user adds items to cart the information of items will be stored in cookies and user will be shown a short report of items selected with total amount. When ever user comes back he/she can see the information of items which were selected previously.
NOTE: after extracting zip file paste all files pther than .java to the appropriate folder of the webserver running on ur pc for storing servlets
